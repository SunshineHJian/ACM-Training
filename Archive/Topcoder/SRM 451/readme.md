题意：
棋盘上有 n 个硬币，问一枚从 (0, 0) 的 baron 最多可以收集多少枚硬币。
baron 行走的规则 (dx, dy) 满足：
>>
dx 严格递增。
dy 恒等于 1。

算法：
动态规划。
对硬币按照 y 坐标从小到大排序，
dp[i][j]：当前在第 i 枚硬币、共收集了 j 枚硬币，最后一步 dx 的最小值。
